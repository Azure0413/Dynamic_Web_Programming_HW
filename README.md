# Dynamic_Web_Programming_HW
此專案為國立成功大學敏求智慧學院開設之動態網頁程式設計，課程中包含前端網頁製作、後端與網頁部屬  
  
chat-app：這是我的final project，我透過php串接mysql製作了一款通訊應用程式，類似於line、messenger等通訊軟體的功能，在ui/ux中也有琢磨許多。  
(由於為動態網頁，因此暫不提供網址)
